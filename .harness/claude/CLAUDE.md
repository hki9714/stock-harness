## Project Overview

**Stock Harness** is a Korean stock market monitoring system that detects
volume surges and buy signals, then sends alerts via Telegram.

- **Runtime**: FastAPI + Uvicorn (no separate WAS needed)
- **Scheduler**: APScheduler (market hours 09:00~15:30 KST, 30min interval)
- **Data**: pykrx (stock prices), Naver Finance discussion crawling
- **AI Analysis**: KR-FinBert-SC sentiment analysis (keyword fallback included)
- **Alerts**: python-telegram-bot
- **Dev Console**: ui/dev_console.html (Claude API integrated)
- **Git**: Auto-commit on file change via watchdog

## Buy Signal Logic

```
Condition 1: Volume surge (3x above 5-day average) + price rising
Condition 2: Price up +5% vs previous close
Condition 3: Naver discussion positive sentiment 65%+ AND 20+ posts/hour

→ ALL 3 conditions met = "BUY" signal → Telegram alert
→ Condition 1 only   = Volume surge alert → Telegram alert
```

Thresholds are configurable in the `stock` section of `config.yaml`.

## How to Run

```bash
# First time setup
python setup/bootstrap.py --run

# Subsequent runs
source venv/bin/activate   # Windows: venv\Scripts\activate
python main.py

# Verify
curl http://localhost:8000/health
curl http://localhost:8000/watch
```

## REST API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/health` | Server status |
| POST | `/check/now` | Trigger immediate signal check |
| GET | `/watch` | List watched tickers |
| POST | `/watch/{code}` | Add ticker |
| DELETE | `/watch/{code}` | Remove ticker |
| POST | `/api/chat` | Claude API proxy |
