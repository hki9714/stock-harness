## 1. 준비물

- Windows 10 / 11 PC
- 스마트폰에 **텔레그램** 앱 설치 (무료, [telegram.org](https://telegram.org))
- 인터넷 연결

그 외 Python, 라이브러리 등은 **이미 프로그램 안에 포함**되어 있습니다.

## 2. 텔레그램 봇 만들기

텔레그램 봇은 알림을 보내주는 계정입니다. 아래 순서대로 무료로 만들 수 있습니다.

### ① BotFather 검색

텔레그램 앱을 열고 검색창에 **`@BotFather`** 를 입력하세요.

```
🔍 검색: @BotFather
```

파란 체크 마크(✅)가 있는 공식 계정을 선택해 대화를 시작합니다.

### ② 봇 생성 명령어 입력

BotFather에게 아래 메시지를 보냅니다:

```
/newbot
```

### ③ 봇 이름 입력

BotFather가 이름을 물어봅니다. 원하는 이름을 입력하세요.

```
예) 주식알림봇
```

### ④ 봇 아이디(username) 입력

`bot`으로 끝나야 하는 영문 고유 아이디를 입력합니다.

```
예) my_stock_alert_bot
```

### ⑤ 토큰 복사

성공하면 아래와 같은 메시지가 옵니다:

```
Done! Congratulations on your new bot. 
You will find it at t.me/my_stock_alert_bot.

Use this token to access the HTTP API:
1234567890:ABCdefGhIJKlmNOpqrSTUvwXYZ
```

**`1234567890:ABC...`** 부분이 **봇 토큰**입니다. 복사해두세요.

## 3. 내 채팅 ID 확인하기

봇이 **나에게** 메시지를 보내려면 내 채팅 ID가 필요합니다.

### ① 방금 만든 봇에게 메시지 보내기

텔레그램에서 내가 만든 봇(@my_stock_alert_bot)을 찾아서 아무 메시지나 보냅니다.

```
안녕
```

### ② ID 확인용 링크 열기

PC 또는 스마트폰 브라우저에서 아래 주소를 엽니다.  
`{봇토큰}` 자리에 위에서 받은 토큰을 넣으세요:

```
https://api.telegram.org/bot{봇토큰}/getUpdates
```

예시:
```
https://api.telegram.org/bot1234567890:ABCdef.../getUpdates
```

### ③ chat id 찾기

열린 페이지에서 아래와 같은 숫자를 찾습니다:

```json
"chat": {
  "id": 987654321,    ← 이 숫자가 채팅 ID입니다
  "first_name": "홍길동"
}
```

**`987654321`** 과 같은 숫자를 복사해두세요.

## 4. 설정 파일 입력하기

프로그램 폴더 안의 **`config.yaml`** 파일을 메모장으로 열어 아래 항목을 입력합니다.  
(첫 실행 시 프로그램이 자동으로 메모장을 열어줍니다)

```yaml
# ── 텔레그램 ─────────────────────────────────────────────────────
telegram:
  bot_token: "1234567890:ABCdefGhIJKlmNOpqrSTUvwXYZ"   ← 2번에서 복사한 봇 토큰
  chat_id: "987654321"                                   ← 3번에서 복사한 채팅 ID
```

### 감시할 종목 변경 (선택)

기본으로 삼성전자, SK하이닉스, NAVER, 현대차가 설정되어 있습니다.  
원하는 종목으로 바꾸려면 종목 코드(6자리)를 입력하세요:

```yaml
stock:
  watch_list:
    - "005930"  # 삼성전자
    - "000660"  # SK하이닉스
    - "035420"  # NAVER
    - "035720"  # 카카오  ← 이런 식으로 추가/변경
```

> 종목 코드는 네이버 증권(finance.naver.com)에서 종목명 검색 후 URL에서 확인할 수 있습니다.

### 저장

입력 완료 후 **파일 → 저장** (또는 Ctrl+S) 하고 메모장을 닫습니다.

## 5. 프로그램 실행하기

프로그램 폴더 안의 **`StockHarness.exe`** 를 더블클릭합니다.

```
📈 Stock Harness v1.0

  서버를 시작합니다...
  주소  : http://localhost:8000
  브라우저가 자동으로 열립니다.
  종료  : 이 창을 닫거나 Ctrl+C
```

- 검은 콘솔 창이 뜨고 잠시 후 브라우저가 자동으로 열립니다
- 텔레그램으로 **"🟢 주식 분석 하네스 시작"** 메시지가 오면 정상 작동 중입니다
- **이 창을 닫으면 프로그램이 종료**되므로 알림을 받는 동안은 창을 유지해주세요

### 자동 시작 설정 (선택)

PC를 켤 때마다 자동으로 실행하려면:

1. `StockHarness.exe` 에 우클릭 → **바로 가기 만들기**
2. 바로 가기를 `Win + R` → `shell:startup` 폴더에 넣기

## 6. 대시보드 사용법

브라우저에서 `http://localhost:8000` 으로 접속하면 대시보드를 볼 수 있습니다.

### 주요 화면

| 메뉴 | 설명 |
|------|------|
| **대시보드** | 감시 종목 현황, 거래량·주가 변동 확인 |
| **스크리닝** | KOSPI 전종목 중 급등 후보 자동 탐색 |
| **백테스트** | 종목별 과거 시그널 성과 및 미래 가격 예측 |

### 종목 추가/삭제

대시보드에서 종목 코드를 직접 입력해 감시 목록을 변경할 수 있습니다.  
변경 내용은 프로그램 재시작 후에도 유지됩니다.

### 알림 조건

아래 3가지 조건이 **모두 충족**될 때 텔레그램으로 매수 시그널 알림이 옵니다:

1. 거래량이 5일 평균의 **3배 이상** + 주가 상승
2. 전일 종가 대비 **+5% 이상** 상승
3. 네이버 증권 게시판 긍정 댓글 **65% 이상** + 시간당 **20건 이상**

거래량 급등만 감지된 경우 별도 급등 알림이 전송됩니다.

## 7. 자주 묻는 질문

### Q. 프로그램 창은 켜져 있어야 하나요?

네. 창을 닫으면 알림이 중단됩니다. 알림을 계속 받으려면 PC와 창을 켜두세요.  
노트북이라면 덮개를 닫아도 실행 유지되도록 전원 설정을 변경해두시면 편합니다.

### Q. 장 마감 후에는 어떻게 되나요?

09:00~15:30 장 시간에만 주식 데이터를 조회하고 알림을 보냅니다.  
장 외 시간에는 스케줄러가 대기 상태로 전환됩니다.

### Q. 텔레그램 메시지가 안 와요

1. 콘솔 창에서 오류 메시지 확인
2. `config.yaml`의 `bot_token`과 `chat_id`가 올바른지 재확인
3. 만든 봇에게 직접 먼저 메시지를 한 번 보냈는지 확인 (3번 단계)

### Q. 방화벽 경고가 뜨면 어떻게 하나요?

Windows Defender 방화벽에서 **"액세스 허용"** 을 클릭하세요.  
로컬 PC에서만 동작하는 서버이므로 안전합니다.

### Q. 종료는 어떻게 하나요?

검은 콘솔 창을 닫거나, 창 선택 후 **Ctrl+C** 를 누르면 종료됩니다.
